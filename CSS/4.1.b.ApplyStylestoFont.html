through

elements pro- vide some default formatting, but you would not want to use those elements throughout a website to control the size of the text. Instead you can use the font-size property. The font-size property accepts relative values that when rendered are relative to the default text size in the browser. The available values are: xx-small, x-small, small, smaller, medium, larger, large, x-large, xx-large.
